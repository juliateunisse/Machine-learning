Introductie


Machine Learning is relatief nieuw en wordt voor steeds meer doeleinden gebruikt. Omdat het relatief nieuw is weten nog veel mensen nog niet precies wat het inhoud. Op deze website wordt uitgelegd wat Machine Learning precies inhoud, waar het op dit moment voor gebruikt wordt en waar het voor zou kunnen worden gebruikt. Ook gaan we dieper in op de geschiedenis van Machine Learning. Machine Learning gaat in de toekomst nog veel meer worden gebruikt en daarom is het belangenrijk dat iedereen weet wat het is en wat je ermee kan doen.
Machine Learning is het bekends als het om robots gaat. Maar er zijn heel veel verschillende soorten robots met verschillende soorten Machine Learning. Daarom gaan we ook dieper in op de verschillende soorten Machine Learning, en verschillende soorten robots met het doeleinden.
